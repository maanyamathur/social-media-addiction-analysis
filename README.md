#Social Media Usage & Addiction Analysis
##Overview
This project analyzes how daily social media usage affects addiction levels, sleep duration, and mental health among students.

##Dataset
705 student records

##Key features:
1.Avg Daily Usage Hours
2.Sleep Hours Per Night
3.Mental Health Score
4.Addiction Score
5.Conflict Frequency
6.Analysis Performed
7.Exploratory Data Analysis
8.Correlation Analysis
9.Multivariate Linear Regression

##Key Results

1.Usage vs Addiction: r = 0.83
2.Usage vs Sleep: r = -0.79
3.Addiction vs Mental Health: r = -0.95
4.Regression Model RÂ² = 0.91

##Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

##Conclusion
Higher social media usage is strongly associated with increased addiction, reduced sleep, and poorer mental health outcomes among students.
