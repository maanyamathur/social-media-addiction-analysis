Social Media Usage & Addiction Analysis
Overview

This project analyzes how daily social media usage affects addiction levels, sleep duration, and mental health among students.

Dataset

705 student records

Key features:

Avg Daily Usage Hours

Sleep Hours Per Night

Mental Health Score

Addiction Score

Conflict Frequency

Analysis Performed

Exploratory Data Analysis

Correlation Analysis

Multivariate Linear Regression

Key Results

Usage vs Addiction: r = 0.83

Usage vs Sleep: r = -0.79

Addiction vs Mental Health: r = -0.95

Regression Model RÂ² = 0.91

Tools Used

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

Conclusion

Higher social media usage is strongly associated with increased addiction, reduced sleep, and poorer mental health outcomes among students.
